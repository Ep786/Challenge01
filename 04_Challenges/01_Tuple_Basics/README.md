### **Tuple Basics**

### Description

This program makes a tuple and prints some parts of it.

### Pseudocode

    Start
    Make a tuple with:
    A number
    A decimal number
    A word
    True or False
    Another small tuple inside  
    Show the whole tuple.  
    Pick the third item and show it.  
    Pick the first item from the small tuple and show it.
