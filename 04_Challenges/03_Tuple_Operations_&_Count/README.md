### **Counting Items**

### Description

This program counts how many times a word appears in a tuple.

### Pseudocode

    Start

    Make a tuple with words like "apple", "banana", and others.  
    
    Ask the user to type a word.  
    
    Count how many times the word appears in the tuple.  
    
    Show the count.
    
    End.
